## Intended use
Advisory ranking of available drivers for the most-urgent pending pickup request
in City2Cruise. The ranking is **advisory**: the dispatch layer applies it only
when `RL_ROUTING_ENABLED=true` and always falls back to geo-distance if the model
is slow/unavailable. Final oversight is human-in-the-loop via the Control Tower
(EU AI Act Art. 14).

## Training data
Fully **synthetic** episodes generated by `rl_service/train_tfm.py` with domain
randomization (Tobin et al., 2017). No personal or production data is used
(confidentiality annex). Reproducible: fixed seed 42, pinned dependencies.

## Limitations
- **Does not surpass the greedy heuristic.** Greedy (nearest-ETA) is near-optimal in
  the current single-step formulation (the target request is fixed each step, so the
  only controllable quantity is the chosen driver's ETA, which greedy minimises).
  Beating greedy requires the anticipatory/batch formulation (future work, §5.2).
- Sim-to-real fidelity is measured against representative reference fixtures, not
  live production telemetry (future work).
- Trained on synthetic data only.
